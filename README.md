# 仓库管理系统 java617

## 项目概述

本项目是一款基于前后端分离架构的WMS（Warehouse Management System）仓库管理系统。该系统面向学生用户，功能全面，覆盖仓库管理主要流程，代码经过严格调试，确保稳定运行。

## 技术栈

- **开发语言**：Java
- **数据库**：MySQL
- **后端框架**：Spring Boot
- **前端框架**：Vue
- **缓存**：Redis

## 功能模块

### 系统看板

- 首页看板：统计仓库、供应商、承运商、客户、预警、过期等信息。

### 仓库管理

- 库存看板：查看和导出仓库、库区和物料数据。
- 出入库管理：包括出库、入库和移库操作，支持批量入库。
- 库存盘点：管理库存盘点流程。

### 物料管理

- 物料分类管理：分类查看物料信息。
- 物料管理：详细物料信息管理。

### 角色管理

- 仓库与库区：增删改查仓库与库区信息。
- 供应商、客户、承运商：各角色相关的增删改查操作。

### 记录与结算

- 库存结算：包括月结和年结。

### 系统工具

- 登陆日志：记录用户登录情况。
- 表单构建、代码生成、系统接口：辅助开发工具。

### 系统监控

- 在线用户、定时任务、数据监控、服务监控、缓存监控：全面监控系统状态。

### 系统管理

- 用户管理、角色管理、菜单管理、部门管理、岗位管理、字典管理、参数设置、通知公告、日志管理：全方位管理后台。

## 运行环境

- 开发环境：Java、MySQL、Spring Boot、Vue、Redis
